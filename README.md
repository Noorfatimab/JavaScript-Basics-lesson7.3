# Lesson 7 – JavaScript Basics (Variables, Functions, DOM Change)

This project introduces JavaScript interactivity with variables and functions.

## Features
- Paragraph with id `displayText` that changes when button is clicked
- Variable `newText` stores the new message
- Function `updateText` changes the paragraph text
- Button click triggers the function

## Files included
- index.html – Webpage structure
- script.js – JavaScript logic
- README.md – Project explanation
